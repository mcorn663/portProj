ExploreIT Knowledge Base

Built an IT Knowledge Base solution to support employees on the available digital tools at the company. The requirement for this project was that the knowledge base needed to be built using SharePoint, it needed to be visually interesting, and it needed to be easy navigation. The end result was a knowledge base that contained over 200 articles, a hierarchical category structure, metadata to organize content, and a site index that created by automation.

SharePoint Learning Journey

Created a SharePoint Guide to help users navigate through an extensive list of SharePoint knowledge base articles. The goal of this project was to create a learning journey for those who wanted to extend their SharePoint knowledge. Based on this, I developed a guide that walked users through ten stages of incremental learning phases that linked to supporting articles and videos.
